# Cloth simulation

Using Java and Processing IDE
Simulating a flag fluttering in the wind


## Installation

- #### Java
- #### Processing IDE: https://processing.org
- #### Toxiclibs: http://toxiclibs.org/docs/p5/toxi/processing/ToxiclibsSupport.html
